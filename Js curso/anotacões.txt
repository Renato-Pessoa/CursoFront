
- JavaScript é uma linguagem de tipagem fraca.

- Evitar usar nomes genéricos e siglas

- A variável do tipk let só é visivel dentro do bloco.

- O efeito de hosting acontece somente com variaveis declaradas com var
e não funciona com let.

- O do/while faz pelo menos uma repeticao, diferente do
while que só fará a repeticao caso a condicao seja
verdadeira.

- O break quebra todo o laço, o continue pula
uma determinada repetiçao
